Roblox Bookmarker Tool

Privacy Policy

Chrome Extension Privacy Notice

Roblox Bookmarker Tool is a browser-based program that helps its users bookmark and conveniently access their favorite Roblox games. The application has been developed while taking into account user privacy issues and uses only necessary permissions.

Information We Collect

Personal information is not collected, sold, nor shared by Roblox Bookmarker Tool. It collects only the information necessary to provide the features of bookmarking.

  • Saved Roblox game bookmarks
  • User extension preferences and settings
  • Extension configuration data

Storage Permission

The extension uses Chrome's storage permission to save user preferences and bookmarked Roblox games locally inside the user's browser.

  • Saved game IDs and bookmark information
  • User interface preferences
  • Extension settings

This information remains stored locally in the user's browser and is not sold or shared with third parties.

Third-Party Services

The Roblox Bookmarker may communicate with services owned by Roblox in order to access information about publicly available games. This extension does not dictate the privacy practices of Roblox, and users should check the privacy policy of Roblox themselves.

Data Security

Reasonable efforts have been made to guarantee the safety of the extension data. Bookmark data of the users is kept locally and is not uploaded to any external servers unless specifically needed for some future feature.

Changes To This Policy

This privacy policy may be updated as new features are added. Any changes will be reflected on this page.

Contact

For questions regarding this privacy policy, please contact the extension developer through the Chrome Web Store support page.
